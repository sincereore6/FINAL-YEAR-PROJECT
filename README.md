📌 Description

An IoT-based system that monitors water flow in real-time, detects leakages, and sends data to a mobile app using the Blynk platform. Designed to reduce water wastage and enable remote monitoring.

⚙️ Features
Real-time water flow monitoring
Leak detection alerts
GPS location tracking
Mobile app integration (Blynk)
Cloud-based data access
🧩 Components
ESP32
Water Flow Sensor
GPS Module (NEO-6M)
Wi-Fi (built-in)
Relay (optional)
🛠️ Software
Arduino IDE
Blynk IoT
🚀 How It Works

Flow sensors measure water usage → ESP32 processes data → GPS adds location → Data sent to Blynk → User monitors via mobile app.

📊 Applications
Smart homes
Irrigation systems
Industrial water monitoring
